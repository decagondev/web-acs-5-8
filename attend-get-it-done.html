What is a Get it Done Session?

Get it Done sessions are live experiences taught by our expert instructors designed to help you prepare the required career artifacts.  

Because Get it Done sessions are structured to focus on different aspects of career artifacts, it is recommended that you come to a session having watched or read the core competency content.

How to Schedule a Get it Done Session in this Unit?

In this unit, you will have access to multiple Get it Done Sessions covering all the career artifact preparation topics. 

SCHEDULE GET IT DONE

Since your career artifacts will be graded in this unit, it is recommended that you attend a Get it Done session to help you finalize your career artifacts.

Get it Done Preparation Checklist:

  • Did you watch or read the core competency content before coming to class?
  • Did you download and make a copy of the resume template?
  • Did you create your LinkedIn and GitHub accounts?
