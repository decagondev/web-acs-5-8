Breadth-First Traversal

Key Concepts

BFS Properties

  • Visits nodes level by level
  • Uses a queue data structure
  • Optimal for shortest path
  • Time complexity: O(V + E)

Applications

  • Shortest path finding
  • Web crawling
  • Social network friends suggestions
  • GPS navigation systems

Connected Components

Understanding Connected Components

Key Concepts

  • Important for graph analysis
  • Used in network partitioning

Implementation Examples

Breadth-First Search Implementation

class Graph {
    constructor() {
        this.adjacencyList = {};
    }

    bfs(startVertex) {
        const queue = [startVertex];
        const visited = new Set();
        const result = [];

        visited.add(startVertex);

        while (queue.length) {
            const currentVertex = queue.shift();
            result.push(currentVertex);

            this.adjacencyList[currentVertex].forEach(neighbor => {
                if (!visited.has(neighbor)) {
                    visited.add(neighbor);
                    queue.push(neighbor);
                }
            });
        }

        return result;
    }
}

Connected Components Implementation

findConnectedComponents() {
    const visited = new Set();
    const components = [];

    for (let vertex in this.adjacencyList) {
        if (!visited.has(vertex)) {
            const component = this.bfs(vertex);
            component.forEach(v => visited.add(v));
            components.push(component);
        }
    }

    return components;
}

Example Usage:

const graph = new Graph();
// Add vertices and edges
graph.addVertex("A");
graph.addVertex("B");
graph.addVertex("C");
graph.addVertex("D");
graph.addVertex("E");

graph.addEdge("A", "B");
graph.addEdge("B", "C");
graph.addEdge("D", "E");

console.log(graph.findConnectedComponents());
// Output: [["A", "B", "C"], ["D", "E"]]
